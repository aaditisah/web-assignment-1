1.String handling functions in C

String handling functions are used to manipulate strings in c programing

  • strlen()-Finds the length of a sting
  • strcpy()-Copies one string to another
  • strcat()-Concatenates two strings
  • strcmp()-compares two strings

2.Shortcut Keys in Windows Operating System

  • Ctrl+C-Copy
  • Ctrl+V-Paste
  • Ctrl+X-Cut
  • Ctrl+Tab-Switch applications
  • Win+D

3.DOS Commands for Files and Driectory Management

  • dir-Display files
  • cd-change directory
  • md-Make directory
  • rd-Remove directory
  • copy-Copy files
  • del-Delete files
